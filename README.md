### Идея
2D top-down аркадный шутер с очками. Хотел сделать зомби и персонажа, но генерировать в GigaChat это было настолько ужасно, что по итогу реализовать это не смог, убил целый час на нормального персонажа, и то GigaChat не может сделать его на прозрачном фоне, а все другие сервисы на ПК в колледже были забанены

### Используемые нейросети
- **GigaChat качинский** для генерации изображений - полный ужас, 1 час пытался получить внятный результат, лучшим из которых стал кадр анимации шага и преобразованный из него IDLE.
- **Qwen** для добавления кадров анимации, с чем он никак не справился (поменять ноги в шаге местами)
- **DeepSeek** для кода
